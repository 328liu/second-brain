# 📚 知识储备

> 从各处收集的知识，整理成可复用的卡片。

```dataview
TABLE
  file.cday AS "收录日期",
  file.folder AS "位置"
FROM #笔记
SORT file.cday DESC
```

---
## 知识卡片

> 每张卡片包含：核心观点 + 原文链接/出处
> 原始图片和文件统一放在对应文件夹中，本页只收录处理后的文本知识。

---
## ➕ 如何添加知识？

新建笔记，在 frontmatter 中加入：
```yaml
tags: [笔记]
source: 原始来源（网页/书籍/文章链接）
```
正文写你整理后的知识点，末尾附上原始链接。
