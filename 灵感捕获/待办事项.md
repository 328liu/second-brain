# 📋 待办事项

```dataview
TABLE
  file.cday AS "日期",
  file.name AS "待办",
  priority AS "紧急程度"
FROM #待办
SORT choice(priority = "紧急", 0, priority = "重要", 1, 2), file.cday DESC
```

---
## ➕ 如何添加待办？

新建笔记，在 frontmatter 中加入 `tags: [待办]` 和 `priority: 紧急/重要/一般`，内容写在正文。
