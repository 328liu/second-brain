# OpenCode 在 Obsidian Vault 中的使用指南

## 工作目录
本 vault 位于 D:\第二大脑

## 核心能力
- 使用 `/obsidian-inbox` 整理收件箱笔记
- 使用 `/wechat-writer` 辅助公众号写作
- 使用 `/everything-search` 全局搜索 vault
- 使用 `/excalidraw-draw` 创建图表

## 知识库结构
- `写作/` - 短篇写作资料（大纲、人设、素材）
- `写作资料/` - 导入的写作素材
- `公众号/` - 公众号文章
- 根目录 - 日常笔记和收件箱

## Obsidian 格式约定
- 使用 `[[wikilinks]]` 链接 vault 内的笔记
- 使用 `![[embed]]` 嵌入其他笔记内容
- 使用 `> [!note]` 标注重点
- 使用 `#tag` 添加标签
- 在文件开头添加 YAML frontmatter（title、tags、aliases、created date）
